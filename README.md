# Hello World Example

A minimal **GitHub starter repository** — the classic first commit when spinning up a new account or testing git workflows.

![Git](https://img.shields.io/badge/Git-Practice-F05032?style=flat-square&logo=git&logoColor=white)

Nothing fancy here — just a placeholder repo for:

- Testing `git clone` / `push` / `pull`
- Verifying GitHub account setup
- Branch and PR experiments

## Quick start

```bash
git clone https://github.com/malimba/simple-hello-worl-example.git
cd simple-hello-worl-example
echo "Hello, world!" >> hello.txt
git add hello.txt && git commit -m "add hello"
git push
```
